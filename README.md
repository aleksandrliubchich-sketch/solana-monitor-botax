Бот для Telegram, который отслеживает крупные покупки токенов на Solana через все DEX и присылает уведомления с:

symbol токена

изображением токена

суммой покупки в SOL

количеством токенов

кошельком покупателя

ссылкой на Solscan

ссылкой на покупку на Axiom.trade

последней новостью Verified Twitter (полный текст + ссылка)

Бот автоматически сводит покупки одного токена за 1 минуту в одно сообщение и не шлёт уведомления чаще 1 раза на токен каждые 5 минут.
